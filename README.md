## Descripción:
Este proyecto continene ejercicios prácticos que se dividen en dos tipos:

1. Fundamentos de python sobre listas, diccionarios, funciones y bucles.
2. Análisis de datos con Pandas como carga el banco de datos, limpieza de los datos, estadísticas, filtros, groupby, columnas calculadas y un análisis libre opcional.

## Dataset:

El dataset salud_mundial.csv contiene 159 países con 13 indicadores de salud pública. 
Los datos incluyen valores nulos intencionalmente para practicar limpieza de datos.

| Columna | Descripción |
|---|---|
| `pais` | Nombre del país |
| `region` | Continente o región geográfica |
| `nivel_ingresos` | Nivel económico: Low, Medium, High, Very High |
| `esperanza_vida` | Esperanza de vida al nacer (años) |
| `gasto_salud_usd` | Gasto per cápita en salud (USD/año) |
| `mortalidad_infantil` | Muertes por cada 1,000 nacidos vivos |
| `medicos_por_1000` | Médicos por cada 1,000 habitantes |
| `poblacion_urbana_pct` | % de población que vive en zonas urbanas |
| `obesidad_pct` | % de la población con obesidad |
| `diabetes_pct` | % de la población con diabetes |
| `tabaquismo_pct` | % de fumadores |
| `camas_hospital_por_1000` | Camas hospitalarias por 1,000 hab. |
| `vacunacion_pct` | % de cobertura de vacunación |

## Cómo ejecutar 
1. Clona el repositorio: `git clone https://github.com/Killstronger/TallerModulo1CD.git` 
2. Instala dependencias: `pip3 install pandas jupyter` 
3. Ejecuta: `jupyter notebook taller_modulo1.ipynb` 


## Hallazgos principales

1. Los paises con menor gasto son mas eficientes a la inversion en la salud.
2. Hay una correlacion entre el gasto en salud y la muerte infantil.
3. Los paises de la region Europa son los que mas gastan en salud.
